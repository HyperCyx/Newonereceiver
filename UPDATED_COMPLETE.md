## 📝 Commands Reference

### Check Versions:
```bash
node --version
pnpm --version
```

### Update Packages:
```bash
cd /workspace
pnpm update --latest
```

### Restart Server:
```bash
pkill -9 -f "next"
cd /workspace
rm -rf .next
PORT=3000 pnpm dev > /tmp/nextjs.log 2>&1 &
```

### Restart Ngrok:
```bash
pkill -9 -f "ngrok"
ngrok http 3000 --log=stdout > /tmp/ngrok.log 2>&1 &
```

### Get Public URL:
```bash
curl -s http://localhost:4040/api/tunnels | grep -o 'https://[^"]*ngrok[^"]*'
```
